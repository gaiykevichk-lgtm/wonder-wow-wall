# Wonder Wow Wall - Финальные требования v1.0

> Документ составлен на основе ТЗ (PDF), загруженного логотипа, скриншотов и указаний по дизайну.

---

## 1. Концепция проекта

**Wonder Wow Wall** -- инновационный сервис отделки стен модульными пластинами.

**Целевая аудитория MVP (v1):** B2C -- физические лица (собственники, арендаторы).

**Ключевые ценности:**
- 100 000+ вариантов цветов и форм
- Быстрый монтаж без шума и пыли (2 часа)
- Визуализация перед покупкой (конструктор, в будущем -- AR)
- Гибкие модели: покупка и подписка
- Кастомизация дизайна

## 2. Брендинг и дизайн

### 2.1 Логотип
- Текст: **WONDER WOW WALL** (sans-serif, bold, uppercase)
- Иконка: **W!** в квадратной рамке (символ стеновой панели)
- Два варианта: серый (монохромный) и серый + зелёный акцент

### 2.2 Цветовая палитра
| Роль | Цвет | HEX |
|------|-------|------|
| Основной фон | Белый | `#FFFFFF` |
| Вторичный фон | Светло-серый | `#F5F5F5` / `#FAFAFA` |
| Текст основной | Тёмно-серый/чёрный | `#2D2D2D` / `#1A1A1A` |
| Текст вторичный | Серый | `#6B7280` / `#9CA3AF` |
| Границы/разделители | Светло-серый | `#E5E7EB` / `#D1D5DB` |
| Акцент (логотип W!) | Зелёный | `#4CAF50` (из лого) |
| Кнопки CTA | Тёмный | `#2D2D2D` |
| Hover/Active | Зелёный | `#4CAF50` |

### 2.3 Принципы дизайна
- **Чёрно-белая база** с серыми оттенками
- **Зелёный акцент** только для: иконки лого, CTA hover, бейджи, статусы
- **Минимализм:** чистые карточки, много воздуха, крупная типографика
- **Рамка в дизайне** (из лого): использовать квадратные рамки/бордеры как дизайн-элемент в карточках, секциях, hero-блоках
- Шрифт: Inter (400, 500, 600, 700, 800)
- border-radius: 12px для карточек, 8px для кнопок/инпутов

- Быстрая загрузка (оптимизация изображений, lazy loading)

### 4.4 Безопасность
- SSL/TLS
- Хеширование паролей
- Соответствие 152-ФЗ

## 5. Технологический стек

### Frontend (Web)
- **React 18+** + TypeScript
- **Vite** -- сборка
- **Ant Design 5** -- UI kit
- **Framer Motion** -- анимации
- **Zustand** -- state management (корзина, auth)
- **React Router 6** -- маршрутизация
- **TanStack Query** -- серверный стейт

### Backend
- **FastAPI** (Python) -- REST API
- **SQLAlchemy** + **Alembic** -- ORM + миграции
- **PostgreSQL** -- БД (SQLite для dev)
- **JWT** -- авторизация
- **S3-compatible** -- хранение изображений

### Инфраструктура
- Docker -- контейнеризация
- Nginx -- reverse proxy
- CI/CD -- автодеплой

## 6. Этапы разработки (из ТЗ, адаптированные)

### Этап 1: MVP (текущий)
Веб-приложение для B2C: каталог, конструктор, заказ, личный кабинет.

### Этап 2: Расширенный функционал
Конструктор кастомных дизайнов (продвинутый), подписки, B2B кабинет, мобильные приложения.

### Этап 3: Масштабирование
CRM-интеграция, склад, аналитика, AR, AI-рекомендации.
