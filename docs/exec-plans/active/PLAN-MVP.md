# Plan: Wonder Wow Wall MVP -- B2C Web-App

> Полный план реализации первой версии веб-приложения для физических лиц.
> Стек: React + Vite + Ant Design + Zustand + FastAPI + SQLite/PostgreSQL.

---

## Фаза 8: Backend API (FastAPI)

### 8.1 Auth API
- [ ] POST `/api/auth/register` -- регистрация
- [ ] POST `/api/auth/login` -- авторизация (JWT)
- [ ] POST `/api/auth/forgot-password` -- восстановление
- [ ] GET `/api/auth/me` -- текущий пользователь

### 8.2 Catalog API
- [ ] GET `/api/designs` -- список дизайнов (фильтры, сортировка, пагинация)
- [ ] GET `/api/designs/:id` -- детали дизайна
- [ ] GET `/api/categories` -- категории
- [ ] GET `/api/designs/:id/reviews` -- отзывы
- [ ] POST `/api/designs/:id/reviews` -- добавить отзыв

### 8.3 Constructor API
- [ ] POST `/api/projects` -- сохранить проект
- [ ] GET `/api/projects` -- мои проекты
- [ ] GET `/api/projects/:id` -- детали проекта
- [ ] DELETE `/api/projects/:id` -- удалить

### 8.4 Orders API
- [ ] POST `/api/orders` -- создать заказ
- [ ] GET `/api/orders` -- мои заказы
- [ ] GET `/api/orders/:id` -- детали заказа

### 8.5 User API
- [ ] GET `/api/users/me` -- профиль
- [ ] PUT `/api/users/me` -- обновить профиль
- [ ] GET `/api/users/me/favorites` -- избранное
- [ ] POST `/api/users/me/favorites/:designId` -- добавить в избранное
- [ ] DELETE `/api/users/me/favorites/:designId` -- убрать из избранного

### 8.6 Other API
- [ ] POST `/api/calculator` -- расчёт стоимости
- [ ] POST `/api/contacts` -- форма обратной связи
- [ ] GET `/api/portfolio` -- портфолио проектов
- [ ] GET `/api/faq` -- FAQ
