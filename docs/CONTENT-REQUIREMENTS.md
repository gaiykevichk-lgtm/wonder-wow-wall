- Файлы: `logo.svg`, `logo.png`, `logo-white.svg`, `favicon.svg`, `icon-192.png`, `icon-512.png`

- Стиль: современный, минималистичный, читаемый на белом и тёмном фоне

## 7. Общие технические требования

### Форматы файлов
| Приоритет | Формат | Когда использовать |
|-----------|--------|-------------------|
| 1 (лучший) | **WEBP** | Основной формат для всех фото |
| 2 (fallback) | **JPG** | Если WEBP недоступен, качество 80–85% |
| 3 (вектор) | **SVG** | Логотипы, иконки, графика |
| 4 (прозрачность) | **PNG** | Только где нужен прозрачный фон |

### Оптимизация
- Все фото — **sRGB** цветовое пространство
- Качество WEBP: **80%** (оптимальный баланс размер/качество)
- Качество JPG: **82–85%**
- Максимальный вес одного файла: **150 КБ**
- Использовать **progressive** JPG для плавной загрузки
- Метаданные EXIF — удалить (уменьшает размер)

## 9. Рекомендации по съёмке

### Панели (текстуры)
- Снимать при **мягком дневном свете** или с 2 источниками (45° с двух сторон)
- Камера перпендикулярно поверхности (без перспективных искажений)
- Показать **фактуру** — лёгкий боковой свет выявляет рельеф
- Нейтральный фон (белый/серый) для вырезки

### Интерьеры (hero, портфолио)
- Широкоугольный объектив (24–35мм)
- Золотое время: утро или закат для тёплого света
- Фокус на стене с панелями, мебель — контекст
- Снимать в RAW для максимальной гибкости цветокоррекции

### Общее
- Все фото — **горизонтально** (кроме квадратных)
- Избегать текста на фото
- Единый цветовой тон (тёплый, уютный)
- Баланс белого: 5500–6500K
