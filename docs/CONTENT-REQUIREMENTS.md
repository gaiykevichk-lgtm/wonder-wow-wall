## 7. Пресеты интерьеров для конструктора

Пресеты — готовые фоны интерьеров, куда пользователь размещает панели прямо в конструкторе
без необходимости загружать своё фото. Интерьер остаётся, панели меняются.

### Как это работает

```
┌─────────────────────────────────────────┐
│  Конструктор                            │
│                                         │
│  ┌─── Выбор фона ───────────────────┐   │
│  │ [Загрузить своё фото]            │   │
│  │ ─── или выберите пресет ───      │   │
│  │ ┌─────┐ ┌─────┐ ┌─────┐         │   │
│  │ │ТВ   │ │Спал.│ │Кухня│ ...     │   │
│  │ │зона │ │     │ │     │         │   │
│  │ └─────┘ └─────┘ └─────┘         │   │
│  └──────────────────────────────────┘   │
│                                         │
│  ┌─── Холст ────────────────────────┐   │
│  │  ╔══════════════════╗            │   │
│  │  ║ Фото интерьера   ║            │   │
│  │  ║ (ТВ-зона)        ║            │   │
│  │  ║                   ║            │   │
│  │  ║  ┌──┬──┬──┐      ║            │   │
│  │  ║  │🪵│🪵│🪵│ ← панели на стене║   │
│  │  ║  ├──┼──┼──┤      ║            │   │
│  │  ║  │🪵│🪵│🪵│      ║            │   │
│  │  ║  └──┴──┴──┘      ║            │   │
│  │  ║       📺          ║            │   │
│  │  ╚══════════════════╝            │   │
│  └──────────────────────────────────┘   │
└─────────────────────────────────────────┘
```

### Что нужно для каждого пресета

Каждый пресет = **3 файла**:

| Файл | Что это | Формат | Размер |
|------|---------|--------|--------|
| **Фото интерьера** | Фоновое фото комнаты | WEBP / JPG | 2048×1365 px (3:2) |
| **Маска стены** | Чёрно-белое изображение: белый = стена, чёрный = мебель/пол/потолок | PNG (grayscale) | 2048×1365 px |
| **Превью** | Миниатюра для выбора пресета | WEBP | 300×200 px |

#### Фото интерьера (основной файл)
| Параметр | Значение |
|----------|----------|
| Формат | WEBP (основной) + JPG (fallback) |
| Размер | **2048×1365 px** (3:2, горизонтальное) |
| Вес файла | до **200 КБ** |
| Качество | 85% WEBP |
| Содержание | Реальный интерьер с видимой стеной |

**Критические требования к съёмке:**
- Стена должна быть **ровная, хорошо освещённая, занимать 30–50% кадра**
- Камера **фронтально к стене** (минимум перспективных искажений)
- Стена — **однотонная или нейтральная** (белая / светло-серая / бежевая)
- Мебель не должна закрывать более 50% стены
- Нет зеркал / стеклянных поверхностей (блики мешают визуализации)
- Освещение **равномерное**, без жёстких теней

#### Маска стены (wall mask)
| Параметр | Значение |
|----------|----------|
| Формат | **PNG** (8-bit grayscale, без альфа) |
| Размер | **2048×1365 px** (совпадает с фото!) |
| Цвета | Белый (`#FFFFFF`) = стена, Чёрный (`#000000`) = не стена |
| Вес файла | до 50 КБ (хорошо сжимается) |

**Как создать маску:**
1. Открыть фото интерьера в Photoshop / Figma
2. Создать слой поверх, залить чёрным
3. Белой кистью закрасить **только область стены** (где можно размещать панели)
4. Исключить: мебель, окна, двери, розетки, плинтуса, потолок, пол
5. Экспортировать как PNG grayscale

```
Пример: ТВ-зона
┌────────────────────────┐
│░░░░░░░░░░░░░░░░░░░░░░░░│ ← потолок (чёрный)
│████████████████████████│ ← стена (БЕЛЫЙ)
│████████████████████████│ ← стена (БЕЛЫЙ)
│████████████████████████│ ← стена (БЕЛЫЙ)
│██████░░░░░░░░██████████│ ← ТВ на стене (чёрный)
│██████░░░░░░░░██████████│
│░░░░░░░░░░░░░░░░░░░░░░░░│ ← тумба (чёрный)
│░░░░░░░░░░░░░░░░░░░░░░░░│ ← пол (чёрный)
└────────────────────────┘
```

  "room": "Гостиная",

### Рекомендации по съёмке пресетов

1. **Объектив:** 24–35 мм (широкоугольник), без дисторсии (или исправить в постобработке)
2. **Штатив обязателен** — камера на высоте ~120 см (уровень глаз стоя)
3. **Камера строго фронтально к стене** — стена должна быть плоской в кадре
4. **Освещение:** все основные светильники включены + дневной свет
5. **Подготовка:** убрать лишнее со стены (провода, мелкий декор), стена должна быть «пустой»
6. **Разрешение:** снимать в максимальном качестве камеры, потом ресайз до 2048px

## 8. Общие технические требования

### Форматы файлов
| Приоритет | Формат | Когда использовать |
|-----------|--------|-------------------|
| 1 (лучший) | **WEBP** | Основной формат для всех фото |
| 2 (fallback) | **JPG** | Если WEBP недоступен, качество 80–85% |
| 3 (вектор) | **SVG** | Логотипы, иконки, графика |
| 4 (прозрачность) | **PNG** | Только где нужен прозрачный фон |

### Оптимизация
- Все фото — **sRGB** цветовое пространство
- Качество WEBP: **80%** (оптимальный баланс размер/качество)
- Качество JPG: **82–85%**
- Максимальный вес одного файла: **150 КБ**
- Использовать **progressive** JPG для плавной загрузки
- Метаданные EXIF — удалить (уменьшает размер)

## 9. Рекомендации по съёмке

### Панели (текстуры)
- Снимать при **мягком дневном свете** или с 2 источниками (45° с двух сторон)
- Камера перпендикулярно поверхности (без перспективных искажений)
- Показать **фактуру** — лёгкий боковой свет выявляет рельеф
- Нейтральный фон (белый/серый) для вырезки

### Интерьеры (hero, портфолио)
- Широкоугольный объектив (24–35мм)
- Золотое время: утро или закат для тёплого света
- Фокус на стене с панелями, мебель — контекст
- Снимать в RAW для максимальной гибкости цветокоррекции
