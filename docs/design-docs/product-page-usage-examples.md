# Секция «В интерьере» на странице товара

## Что добавлено

На страницу каждого товара добавлена секция **«В интерьере»** — галерея реальных интерьерных фотографий, показывающая панель в контексте жилых пространств. Вдохновлено разделом «iPad in the wild» на apple.com/ipad-pro.

---

## Архитектура

### Изменения в типах (`model/types.ts`)

Добавлено новое поле в интерфейс `PanelProduct`:

```typescript
usageExamples?: {
  room: string;    // Название помещения: «Гостиная», «Ванная», и т.д.
  image: string;   // URL фото (Unsplash, 1200×800)
  caption: string; // Короткий Apple-style текст под фото
}[];
```

Поле опциональное — если не заполнено, секция не отображается.

### Дизайн-решения в стиле Apple

1. **Overlay gradient** — тёмный градиент снизу вверх на каждой фотографии (rgba 0→0.55 → transparent)
2. **Room pill** — frosted glass эффект (`backdrop-filter: blur(12px)`, `rgba(255,255,255,0.15)`)
3. **Caption on image** — белый текст 21px/500 с `letterSpacing: -0.01em` поверх фото
4. **Hover zoom** — `whileHover: { scale: 1.03 }` с `transition: 0.8s ease` через framer-motion
5. **Scroll animation** — `whileInView` с `once: true`, появление снизу (`y: 24 → 0`)
6. **Border radius** — главное фото 28px, вторичные 24px
7. **Gap** — 12px между карточками (плотнее чем стандартный grid)
8. **Responsive** — secondary grid использует `auto-fit, minmax(300px, 1fr)` — на мобильных переходит в 1 колонку

## Контент-гайдлайн для копирайтера

Стиль подписей к фото (Apple-style):
- Короткие предложения (1-2 строки)
- Поэтический/образный язык, не описательный
- Никаких «здесь вы можете...» — только образ
- Точка в конце каждого предложения

**Примеры:**
- ✅ «Тёплая текстура дуба создаёт атмосферу живого дерева. Каждая деталь в гармонии.»
- ✅ «Мрамор — это вечный разговор о красоте. Бесконечно элегантно.»
- ❌ «Дубовая классика хорошо смотрится в гостиной с диваном.»

## Как добавить примеры для новых продуктов

1. Найти 2-3 Unsplash фото в формате 1200×800, релевантных помещениям в `product.room[]`
2. Добавить поле `usageExamples` в объект продукта в `data.ts`:

```typescript
usageExamples: [
  {
    room: 'Гостиная',
    image: 'https://images.unsplash.com/photo-XXXXXXXXXX?w=1200&h=800&fit=crop',
    caption: 'Короткий поэтический текст. Второе предложение.',
  },
  // ...
],
```

Секция появится автоматически — код в `ProductPage.tsx` уже обрабатывает `usageExamples`.
